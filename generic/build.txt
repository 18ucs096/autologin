	
To Build packages

All scripts carry two parameters , 

	1st arg is environment {dev|prod}
	2nd arg is (all|debug)  , {debug} create a expanded delivery under dist/build/autologin-{browser}
							  {all} could package xpi for firefox & zip for chrome& opera  , Safari should be built after loading into safari extension builder
							  

sample build commands
							  
tools/make-firefox.sh dev all		
tools/make-chromium.sh dev all
tools/make-opera.sh dev all

	There is a shell script & python script used for each browser.
